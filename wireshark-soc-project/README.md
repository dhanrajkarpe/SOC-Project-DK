Wireshark SOC Project

Overview
This project demonstrates network traffic analysis using Wireshark to detect suspicious activities like port scanning and abnormal traffic.

Tool
Wireshark

Steps
1. Capture network traffic  
2. Apply filters (TCP, DNS, SYN)  
3. Analyze packets  
4. Detect suspicious activity  

Filters Used
tcp
dns
tcp.flags.syn == 1

Detection
Port scanning (SYN packets)  
Suspicious IP traffic  
DNS activity
