
# Phishing Email Analysis & Investigation

## Project Overview
This project demonstrates how a SOC analyst investigates a phishing email.
The goal is to analyze the email header, identify malicious indicators,
and determine whether the email is part of a phishing attack.

## Tools Used
- Email Header Analyzer
- VirusTotal
- WHOIS Lookup
- URLScan.io
- MXToolbox

## Investigation Steps

### 1. Email Header Analysis
Analyze the email headers to identify the originating IP address and mail servers.

### 2. URL Inspection
Extract suspicious URLs and check them using VirusTotal and URLScan.

### 3. Domain Reputation Check
Use WHOIS and reputation services to analyze the sender domain.

### 4. Indicator of Compromise (IOC) Extraction
Identify malicious IPs, domains, and URLs.

### 5. Final Verdict
Determine whether the email is malicious or safe.
